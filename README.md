# Music Creation Skills

Reusable skills for local/procedural music creation.

## Included skills

- `strudel-music` — write runnable Strudel live-coding sketches, especially pop-punk, punk, grunge, and browser-based music patterns.
- `local-music-generation` — generate cheap/local procedural WAV/MP3 tracks with arrangement, humanization, and publishing workflow.

## Install

Install the Strudel skill:

```bash
npx skills add https://github.com/slapash/music-creation-skills --skill strudel-music
```

Install the local rendering skill:

```bash
npx skills add https://github.com/slapash/music-creation-skills --skill local-music-generation
```

## Notes

These are pragmatic music-sketching skills, not a promise of radio-ready vocals or Suno-level full productions. They are meant for fast iteration, background music, rough punk/pop-punk sketches, and browser-live-coding workflows.
